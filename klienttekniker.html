Klienttekniker


HTML

HTML är en förkortning av HyperText Markup Language och är en av webbens grundläggande byggstenar. HTML utgör själva strukturen för alla internets webbsidor i form av rubriker, styckeindelning, att särskilja visst innehåll från annat och mycket mer. HTML definieras som ett märkspråk och inte ett programmeringsspråk, eftersom HTML-kod endast har i uppgift att ge struktur.


CSS

CSS är en förkortning av Cascading Style Sheets och är en av webbens tre grundläggande byggstenar tillsammans med HTML och Javascript. Man kan säga att CSS är webbens ansikte utåt eftersom det sköter allt som har med själva designen att göra. Med hjälp av CSS kan man åstadkomma allt från att välja typsnitt på text till att göra stilrena och mer avancerade animationer.


JavaScript

Javascript är en av webbens tre grundläggande byggstenar tillsammans med HTML och CSS. Om HTML utgör strukturen och CSS utgör designen så kan man påstå att Javascript utgör funktionaliteten på internets webbsidor. Ett väldigt vanligt användningsområde för Javascript är att förutbestämma vad som ska hända när en användare interagerar på olika sätt på en webbsida.
